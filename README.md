This application runs in Visual Studio.

If you want to run it in VS Code, you need to install the .NET SDK first, and then follow the same steps below.

Step 1: Go to the Project Folder
Open the terminal and navigate to the folder where the application file exists.

Step 2: Run the Application
dotnet run -- --hotels hotels.json --bookings bookings.json
